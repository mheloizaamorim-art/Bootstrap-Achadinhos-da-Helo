# 🛍️ Achadinhos da Helô - Projeto Bootstrap

Este projeto é um site de curadoria de produtos (e-commerce fictício) desenvolvido como parte de um exercício prático para a disciplina de **Desenvolvimento Web Front-End**.

O objetivo principal foi aplicar os conceitos do framework **Bootstrap 5**, focando em responsividade, componentes de interface e personalização de temas.

---

## 🚀 Funcionalidades

- **Home:** Exibição de produtos principais utilizando o sistema de `Cards` do Bootstrap.
- **Cadastro de Itens:** Formulário de cadastro com validação visual, estilizado com ícones e bordas personalizadas.
- **Sobre Nós:** Página institucional contando a história da marca com layout responsivo.
- **Navegação:** Navbar fixa (`sticky-top`) com barra de busca integrada e efeitos de hover.

## 🛠️ Tecnologias Utilizadas

* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) (Com variáveis personalizadas para a paleta de cores)
* [Bootstrap 5](https://getbootstrap.com/)
* [Bootstrap Icons](https://icons.getbootstrap.com/)

## 🎨 Identidade Visual

A paleta de cores principal utiliza o tom `#6f42c1` (Roxo), buscando transmitir uma imagem moderna e criativa para a marca "Achadinhos da Helô".
